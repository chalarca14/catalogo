# 📱 Catálogo de Smartphones

Bienvenido al repositorio **catalogo**, una tienda en línea que ofrece una amplia variedad de smartphones de marcas reconocidas como Samsung, iPhone y Xiaomi. Este proyecto tiene como objetivo proporcionar a los usuarios una experiencia de compra intuitiva y moderna.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura del contenido.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Funcionalidades interactivas y dinámicas.

## 🔍 Características Principales

- Catálogo organizado por marcas: Samsung, iPhone y Xiaomi.
- Filtros de búsqueda para facilitar la navegación.
- Detalles de productos con imágenes, descripciones y precios.
- Funcionalidad de carrito de compras (en desarrollo).

## 🚀 Cómo Ejecutar el Proyecto

1. Clona este repositorio en tu máquina local:

   ```bash
   git clone https://github.com/chalarca14/catalogo.git
````

2. Navega al directorio del proyecto:

   ```bash
   cd catalogo
   ```

3. Abre el archivo `index.html` en tu navegador preferido.

   ```bash
   open index.html
   ```

   o en Windows:

   ```bash
   start index.html
   ```
